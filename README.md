# 90DaysOfDevops
# A structured hands-on journey through DevOps, Cloud Engineering, Infrastructure Automation, and Platform Engineering.

---

## Overview

This repository documents my continuous learning and practical implementation of modern DevOps practices and cloud-native technologies.

The objective of this journey is to strengthen core infrastructure knowledge while building production-oriented skills across automation, CI/CD, containerization, orchestration, monitoring, and cloud platforms.

The repository will evolve progressively with hands-on labs, notes, configurations, architecture concepts, troubleshooting scenarios, and mini-projects.
